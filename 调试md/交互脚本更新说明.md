# 交互脚本更新说明

## 更新内容

### 新增功能
1. **最大结果数量设置** - 现在可以选择设置最大搜索结果数量
2. **动态配置调整** - 可以在运行过程中修改各种参数
3. **改进的菜单界面** - 更清晰的选项布局

### 主要修改

#### 1. 默认最大结果数量
- **修改前**: 30篇
- **修改后**: 100篇（可根据需要调整）

#### 2. 新增菜单选项
```
原菜单 (5个选项):
1. 使用默认搜索词开始搜索
2. 输入新的搜索词  
3. 查看详细搜索词
4. 启用/禁用全文提取功能
5. 退出

新菜单 (6个选项):
1. 使用默认搜索词开始搜索
2. 输入新的搜索词
3. 查看详细搜索词
4. 设置最大结果数量        ← 新增
5. 启用/禁用全文提取功能
6. 退出程序               ← 新增
```

#### 3. 最大结果数量设置功能
- 选择选项4后，系统会显示当前值
- 可以输入新的数值进行修改
- 支持直接按Enter保持当前值
- 输入验证：只接受正整数

#### 4. 配置显示改进
现在启动时会显示：
```
当前配置:
默认搜索词: ...
最大结果数: 100          ← 显示当前值
全文提取功能: 关闭       ← 新增显示
邮箱: your_email@example.com
```

### 使用方法

#### 运行程序
```bash
python pubmed.py
```

#### 设置最大结果数量示例
1. 启动程序
2. 输入 `4` 选择"设置最大结果数量"
3. 系统显示: "当前最大结果数量: 100"
4. 输入新值 (例如: 150) 或直接按Enter保持当前值
5. 系统确认: "✅ 最大结果数量已更新为: 150"
